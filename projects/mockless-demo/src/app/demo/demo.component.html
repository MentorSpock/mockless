Mockless Demo

Angular library for intelligent HTTP request recording and replay

🎯 What is Mockless?

Mockless provides seamless API mocking during development and testing by automatically recording HTTP requests and their responses, then replaying them when needed.

📡 Automatic Recording: Intercepts all HTTP requests and responses
🔄 Request Replay: Automatically replays recorded responses for matching requests
❌ Error Handling: Records and replays both successful responses and error conditions
✏️ Editable Mocks: Modify recorded responses through an intuitive JSON editor
🔧 Toggle Mode: Easy enable/disable functionality via localStorage
🧪 Record Viewer: View the panel
